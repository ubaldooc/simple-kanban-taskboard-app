

-------------- PARA INICIAR EL PROYECTO --------------------

1. Abre el archivo llamado "iniciar-proyecto" , es un bat que se encarga de iniciar el 
	servidor de vite para el frontend y node para el backend. Actualmente estoy usando fnm para 
	el control de versiones de node por lo que si no te funciona tendras que iniciar el proyecto
	manualmente usando los comandos de la terminal "npm run dev" en la carpeta del frontent y otro en la
	carpeta del backend


2. Debes configurar las variables de entorno con tus propias configuraciones: 
	
	Para el frontend:
	-- VITE_API_BASE_URL (url para conectar con el backend). 
	-- VITE_GOOGLE_CLIENT_ID (para la funcionalidad de iniciar sesion). 
	
	Para el backend:
	-- PORT (puerto del servidor del backend).
	-- FRONTEND_URL (servidor del frontend).
	-- MONGO_URI (direccion de la base de datos de mongoDB).

	-- JWT_SECRET (autenticacion de tokens).
	-- GOOGLE_CLIENT_ID (para funcionalidad de iniciar sesi칩n).
	-- GOOGLE_CLIENT_SECRET (para funcionalidad de iniciar sesi칩n).

	-- EMAIL_USER (para correos autom치ticos).
	-- EMAIL_PASS (para correos autom치ticos).

	-- CLOUDINARY_CLOUD_NAME (para funcionalidad de wallpapers).
	-- CLOUDINARY_URL (para funcionalidad de wallpapers).
	-- CLOUDINARY_API_KEY (para funcionalidad de wallpapers).
	-- CLOUDINARY_API_SECRET (para funcionalidad de wallpapers).
	-- DEFAULT_WALLPAPER_URL "(url del wallpaper por defecto, desde Cloudinary)."
